# Yelp - Website Project

## Description

Sample website using the following web dev technologies:
* HTML5
* CSS3
  * Bootstrap 4
  * Semantic UI
* Javascript
  * jQuery
* Node.js
  * NPM
  * Express.js
  * Mongoose
  * Authentication (Passport.js)
* Database
  * MongoDB
* RESTful Routing
